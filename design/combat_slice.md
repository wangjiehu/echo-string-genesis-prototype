# MVP战斗切片设计

目标：让动作系统服务语义改写，而不是把语义系统做成战斗暂停菜单。

## 1. 战斗原则

1. 玩家必须能靠基础动作活下来。
2. 玩家必须通过语义改写打得更聪明、更快。
3. Boss不能被纯普攻高效击败。
4. 每次语义窗口都要短、清楚、有收益。
5. 战斗中一次只展示最关键的1到2个可改写槽位。

## 2. 玩家动作集

| 动作 | 作用 | MVP要求 |
| --- | --- | --- |
| 轻攻击 | 稳定输出和连段起手 | 3段连击即可 |
| 重攻击 | 破防、击退、触发破裂窗口 | 前摇明显，收益明确 |
| 闪避 | 规避伤害和调整站位 | 不绑定语义系统 |
| 格挡 | 降低伤害 | 非完美格挡不打开语义窗口 |
| 完美弹反 | 打断攻击并打开破绽 | MVP核心战斗入口 |
| 扫描 | 读取敌人关键词缀 | 移动中可慢速使用 |
| 剥离/注入 | 改写敌人槽位 | 只在破绽或低威胁窗口中高效执行 |

## 3. 语义破绽窗口

### 3.1 触发方式

- 完美弹反
- 重攻击打空敌人韧性
- 组合反应命中
- 敌人大招后摇

### 3.2 窗口规格

| 敌人类型 | 窗口时长 | 可显示槽位 | 允许操作 |
| --- | ---: | --- | --- |
| 字壳游兵 | 2.5秒 | 结构、温度 | 剥离、注入 |
| 缀刺兽 | 2.0秒 | 质量、介质 | 剥离、注入 |
| 残章守卫 | 1.8秒 | 当前阶段相关槽位 | 剥离、注入、反应触发 |

### 3.3 失败处理

窗口结束后，如果玩家正在选择词缀，不硬暂停游戏，只保留0.5秒缓冲。这样避免战斗被频繁打碎。

## 4. 敌人设计

### 4.1 字壳游兵

**定位**：基础近战兵，教会战斗扫描。

**默认状态**：

- 结构：坚固或脆弱
- 行为：追猎

**招式**：

1. 横劈：慢速近战，适合弹反。
2. 盾撞：小位移攻击，坚固状态下更强。
3. 断句停顿：攻击后短暂停顿，适合扫描。

**语义解法**：

- 剥离坚固，降低减伤。
- 注入脆弱，制造高伤窗口。
- 注入冻结，短暂控制。

**反思**：  
游兵的职责是教学，不要设计成复杂敌人。它应该让玩家第一次觉得“我看懂它了”。

### 4.2 缀刺兽

**定位**：冲撞敌人，教会质量和介质反制。

**默认状态**：

- 质量：沉重
- 温度：过热
- 行为：追猎

**招式**：

1. 蓄力冲撞：沉重状态下伤害高，但转身慢。
2. 热刺甩尾：近身范围攻击，带轻微燃烧。
3. 地面刨击：制造短暂热区。

**语义解法**：

- 注入轻盈，降低冲撞威力并增加击飞。
- 注入潮湿，和过热触发蒸汽爆发。
- 用排斥打断冲撞。

**反思**：  
缀刺兽是第一个“词缀既有好处也有坏处”的敌人。沉重让它强，也让它笨；过热让它危险，也让它能被潮湿反应。

### 4.3 残章守卫

**定位**：MVP小Boss，综合考核。

**Boss规则**：

- 结构槽开局锁定。
- 每次环境反应命中核心，护文层减少1层。
- 护文层为0时，结构槽开放5秒。
- 玩家注入脆弱并重击可进入处决。

**阶段一：坚固护文**

- 招式：宽幅剑扫、盾墙推进。
- 可用反制：过热 + 潮湿蒸汽爆发，或重攻击打护文裂口。
- 教学重点：锁定槽位不是不能互动，而是要先破保护层。

**阶段二：沉重冲撞**

- 招式：三段冲撞、地面震荡。
- 可用反制：轻盈削弱冲撞，排斥打断路线。
- 教学重点：质量词缀影响动作行为。

**阶段三：过热暴走**

- 招式：热浪环、追踪火线。
- 可用反制：潮湿触发蒸汽爆发，冻结制造短控制。
- 教学重点：组合反应是最有效破绽入口。

**胜利流程**：

1. 打掉三层护文。
2. 结构槽开放。
3. 注入脆弱。
4. 重击核心。
5. Boss解体，打开修正文库。

**反思**：  
Boss阶段不能太多。MVP里三阶段已经是上限，每阶段只强调一个词缀逻辑。

## 5. 战斗数值初稿

| 指标 | 建议值 | 说明 |
| --- | ---: | --- |
| 玩家生命 | 100 | 允许犯错，但不能无视攻击 |
| 游兵普通攻击 | 10 | 教学敌人 |
| 缀刺兽冲撞 | 24 | 应明显危险 |
| Boss重击 | 28 | 要求闪避或弹反 |
| 普通轻攻击 | 6 | 稳定输出 |
| 普通重攻击 | 16 | 触发破裂窗口收益 |
| 脆弱伤害倍率 | 1.8 | 和数据表一致 |
| 坚固伤害倍率 | 0.55 | 和数据表一致 |
| 蒸汽爆发伤害 | 12 | 主要价值是击退和致盲 |

## 6. 战斗验收

- 玩家第一次遇到游兵时，至少50%测试者会主动扫描。
- 缀刺兽不能被站桩轻攻击轻松击败。
- Boss战中，玩家平均至少使用4次语义操作。
- 蒸汽爆发触发后，玩家能看清敌人轮廓。
- 完美弹反窗口不能窄到只服务高手，首版可偏宽。

## 7. 当前反思

最大风险是“扫描/改写让战斗停顿太多”。首版不要追求华丽语义连招，而是把改写集中到破绽窗口。等玩家确认这个节奏舒服后，再考虑后期的攻击句法改写。
